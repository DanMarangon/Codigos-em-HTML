Web Semântica

A web semântica é um conceito que visa organizar os dados na web, permitindo que máquinas os compreendam e interajam com ela (web). Para isso usamos várias tags semânticas que facilitarão o trabalho dos mecanismos de busca.

Neste exemplo estamos usando a web semântica na prática, use um bloco de notas para acessar o documento html e ver como essa página foi construída
