Estrutura de um documento HTML

A Estrutura de um documento HTML é definida pela declaração !DOCTYPE HTML, pelas tags html, head e body.

Essa estrutura é interpretada pelo navegador que analisa cada tag e renderiza para o usuário.

A tag head é utilizada para configurações do navegador e cosequentemente do documento HTML, as informações contidas dentro dessa tag são invisíveis para o usuário, com exceção do título que ficará visível na parte superior da página

Já a tag body é utilizada para exibir toda a parte visível para o usuário e nela teremos outras tags que também serão utilizadas pelo navegador

Tag HTML, essa tag é a raíz que contem todas as outras tags do documento HTML, delimitando a totalidade do documento

Por fim, a instrução DOCTYPE html, que tem a finalidade de indicar ao navegador qual a versão do html que será utilizado. Ela deve vir na primeira linha do documento
